<h1>Academia de Ciencias Naturales</h1>
<p>Sitio web multipágina de cursos desarrollado con HTML y CSS. El sitio presenta una plataforma ficticia, con una sección de historia, misión, valores, cursos y contacto.</p>

<h3>Demo:</h3>

<h3>Tecnologías Utilizadas:</h3>
<ul>HTML</ul>
<ul>CSS</ul>

<h3>Estructura del Proyecto:</h3>
<ul>Archivo para Sitio: contiene imagenes y video usado de fondo.</ul>
<ul>Cursos: imagenes que no fueron utilizadas</ul>
<ul>Archivos HTML: index, MetodologiadelaEnsenanza, Evolucion, BotanicaparaTodos.</ul>
<ul>Archivos CSS: inicio, metodologia, evolucion,botanica</ul>

<h3>Funcionalidades:</h3>
<ul>Navegación entre páginas mediante enlaces.</ul>
<ul>Listado de cursos</ul>
<ul>Sección de Contacto</ul>

<h3>Objetivo del Proyecto:</h3>
<p>Originariamente el proyecto era para una academia de enseñanza de ciencias naturales que estaba armando, pero dados los tiempos, compromisos y nuevas metas decidí dejarlo como un 
proyecto de práctica de desarrollo Frontend utilizando únicamente HTML y CSS.</p>
